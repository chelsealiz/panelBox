Panel Options

Here are some of the options that you can use when creating a panel

  • Width- Change the panel's width. This can be in percentages or pixels. (default- 100%)
  • Height- Adjust the height of the panel. This can be in percent or pixels. (default- 50%)
  • Direction- Which direction does the panel come from? (default- top)
  • Background- Panel's background color (default- white/#FFFFFF)
  • Text Color- Text color for the panel (default- black/#000000)

There are so many more things you can do with these panels. Dismiss this panel and read on!

PanelBox Introduction

Introduction and Features

This jQuery library allows you to create customizable, dismissable information panels. You don't even have to use JavaScript or jQuery. There are tons of features and it's super easy to implement.
